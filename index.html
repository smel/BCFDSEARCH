BCFD SEARCH

  • $name (Compatible with all except member leave and member ban) - This mentions the user who sent the message. NOTE: If you use it in the channel send part of an event where the user leaves you will receive an invalid user mention.
  • $channel (Compatible with Message Received) - Adds the channel name to the message.
  • $server (Not compatible with Private Message Received) - Adds the server name to the message
  • $message (Compatible with Message Received and Private Message Received) - This adds the message content of the message sent.
  • $rollnum(min,max) (Compatible with all) - This generates a random number where you replace min with the smallest number and max with the largest one. For example, if you made a dice command that uses $rollnum(1,6) it would show replace where you put the $rollnum(1,6) with the result. If your command was to reply with "$rollnum(1,6) is the roll of your dice." it would appear as below. But if you made a mistake in your command and put in a number larger than the max in the min spot like "$rollnum(55,12)" You could get an error that appears like so.
  • $date (Compatible with all) - The current date.
  • $avatar (Compatible with all except member leave and member ban) - The current avatar of the user who called the command.
  • $defaultAvatar (Compatible with all except member leave and member ban) - The default avatar of the user.
  • $memberCount (Not compatible with pm recieved) - Gives the number of members currently in the server.
  • $serverCreateTime (Not compatible with pm recieved) - Gives the time the server was created.
  • $messageAfterCommand (Compatible with Message Received and Private Message Received) - gives all raw message content after command.
  • $commandCount - gives the number of commands bot has.
  • $namePlain - name of user who called command without mention.
  • $serverIcon (Not compatible with pm recieved) - icon of the server
  • $serverBanner (Not compatible with pm recieved) - banner of the server, if it exists
  • $serverDescription (Not compatible with pm recieved) - description of the server, if it exists
  • $serverSplash (Not compatible with pm recieved) - splash of the server, if it exists
  • $botAvatar - avatar of the bot
  • $botName - name of the bot
  • $mentionedAvatar (Only message recieved) - avatar of mentioned user
  • $mentionedName (Only message recieved) - name of mentioned user in a mention
  • $mentionedNamePlain (Only message recieved) - name of a mentioned user in plain format
  • $random{} - Used to select a random message or link for embed images. Used like $random{abc|def|ghi} where | is the seperator between choices to be picked.
  • $discriminator - discriminator of command caller
  • $tag - tag of command caller
  • $timeCreated - account creation time of command caller
  • $ID - id of command caller
  • $serversSharedWithBot - number of servers shared with command caller
  • $channelID (Only message recieved or react add) - id of channel where message is
  • $channelCreateDate (Only message recieved or react add) - date of the creation of the channel
  • $channelAsMention (Only message recieved or react add) - mention of channel
  • $botDiscriminator - discriminator of the bot
  • $botTag - tag of bot
  • $botDefaultAvatar - default avatar of bot
  • $botTimeCreated - creation time of bot
  • $botID - id of bot
  • $ping - ping to gateway
  • $inviteURL - url to invite the bot
  • $serverCount - amount of servers bot is in
  • $allMemberCount - amount of users in all the bot is in
  • $mentionedDiscriminator (Only message recieved) - discriminator of mentioned user
  • $mentionedTag (Only message recieved) - tag of mentioned user
  • $mentionedIsBot (Only message recieved) - is the mentioned user a bot?
  • $mentionedDefaultAvatar (Only message recieved) - default avatar of mentioned user
  • $mentionedTimeCreated (Only message recieved) - account creation date of mentioned user
  • $mentionedID (Only message recieved) - id of mentioned user
  • $mentionedServersSharedWithBot (Only message recieved) - how many servers the mentioned user shares with the bot.
